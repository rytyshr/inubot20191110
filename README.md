# 犬bot
### LINE用のbotです。犬の画像を送ります。
- メッセージを送ると犬の画像を返す、ただそれだけです
- メッセージは何でも良いです
- スタンプ・画像・位置情報では返事しません
- 通信状況の影響により画像が返らない場合がありますが、その際は時間をあけてメッセージを送ってください
### 使ったもの
- heroku
- node.js
- LINE Messaging API
- Dog API
